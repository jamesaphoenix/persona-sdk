# Getting Started

This page will walk you through the basics of creating personas, using distributions, and optimizing prompts. For complete documentation, refer to the API Reference.

## Installation

```bash
npm install @jamesaphoenix/persona-sdk
# or
yarn add @jamesaphoenix/persona-sdk
# or
pnpm add @jamesaphoenix/persona-sdk
```

## Quick Start

```typescript
import { PersonaBuilder, PersonaGroup, NormalDistribution } from '@jamesaphoenix/persona-sdk';

// Create a single persona
const persona = PersonaBuilder.create()
  .withName('Alice Johnson')
  .withAge(28)
  .withOccupation('Software Engineer')
  .withSex('female')
  .build();

// Create a group with distributions
const group = await PersonaGroup.generate({
  name: 'Tech Workers',
  size: 100,
  attributes: {
    age: new NormalDistribution(32, 5),
    occupation: 'Developer',
    sex: 'other'
  }
});
```

## Core Concepts

### PersonaBuilder

The PersonaBuilder provides a fluent API for creating individual personas:

```typescript
const persona = PersonaBuilder.create()
  .withName('John Smith')
  .withAge(35)
  .withOccupation('Product Manager')
  .withSex('male')
  .withAttribute('department', 'Engineering')
  .build();
```

### PersonaGroup

PersonaGroup manages collections of personas and provides statistical analysis:

```typescript
const group = new PersonaGroup('Customers');

// Generate from distributions
group.generateFromDistributions(1000, {
  age: new NormalDistribution(42, 12),
  occupation: 'Professional',
  sex: 'other'
});

// Get statistics
const ageStats = group.getStatistics('age');
console.log(`Average age: ${ageStats.mean}`);
```

### Statistical Distributions

Use distributions to create realistic variation:

```typescript
import { 
  NormalDistribution, 
  UniformDistribution, 
  CategoricalDistribution 
} from '@jamesaphoenix/persona-sdk';

// Age follows normal distribution
const age = new NormalDistribution(35, 8).withMin(18).withMax(65);

// Income has uniform range
const income = new UniformDistribution(30000, 150000);

// Department has specific probabilities
const department = new CategoricalDistribution([
  { value: 'Engineering', probability: 0.4 },
  { value: 'Sales', probability: 0.3 },
  { value: 'Marketing', probability: 0.3 }
]);
```

## AI-Powered Features

### Structured Output Generation

Generate type-safe insights from your personas:

```typescript
import { StructuredOutputGenerator } from '@jamesaphoenix/persona-sdk';
import { z } from 'zod';

const generator = new StructuredOutputGenerator();

const InsightsSchema = z.object({
  segments: z.array(z.object({
    name: z.string(),
    size: z.number(),
    characteristics: z.array(z.string())
  })),
  recommendations: z.array(z.string())
});

const insights = await generator.generate(
  group,
  InsightsSchema,
  "Analyze market segments and provide recommendations"
);

console.log(insights.data.segments);
```

### Distribution Selection

Let AI choose appropriate distributions:

```typescript
import { DistributionSelectorLangChain } from '@jamesaphoenix/persona-sdk';

const selector = new DistributionSelectorLangChain();

const result = await selector.selectDistributions({
  description: "Tech startup employees in Silicon Valley",
  requirements: ["Age should skew younger", "High income variation"],
  attributes: ['age', 'income', 'experience']
});

// Use AI-selected distributions
const group = new PersonaGroup('Startup Team');
group.generateFromDistributions(100, result.distributions);
```
