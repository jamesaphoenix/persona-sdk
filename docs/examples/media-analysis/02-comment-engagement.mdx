## The Setup: Know Your Audience Segments ðŸŽ­

```typescript
import { PersonaGroup, StructuredOutputGenerator } from '@jamesaphoenix/persona-sdk';
import { NormalDistribution, BetaDistribution } from '@jamesaphoenix/persona-sdk/distributions';
import { z } from 'zod';

// Create a diverse audience with different engagement patterns
const diverseAudience = await PersonaGroup.generate({
  size: 5000,
  segments: [
    { 
      occupation: 'tech',
      weight: 0.3,
      attributes: {
        comment_likelihood: new BetaDistribution(3, 2), // Higher engagement
        controversy_tolerance: new NormalDistribution(7, 1.5)
      }
    },
    { 
      occupation: 'academic',
      weight: 0.2,
      attributes: {
        comment_likelihood: new BetaDistribution(2, 3), // Thoughtful but less frequent
        controversy_tolerance: new NormalDistribution(8, 1)
      }
    },
    { 
      occupation: 'general',
      weight: 0.5,
      attributes: {
        comment_likelihood: new BetaDistribution(1, 4), // Mostly lurkers
        controversy_tolerance: new NormalDistribution(5, 2)
      }
    }
  ]
});
```

## Analyze Content for Maximum Engagement ðŸš€

```typescript
const contentPiece = {
  topic: "AI Ethics in 2025",
  complexity: "high",
  controversial: true,
  tone: "thought-provoking",
  length: 1200, // words
  hasQuestions: true,
  hasPoll: false,
  platform: "LinkedIn"
};

const generator = new StructuredOutputGenerator();

const EngagementSchema = z.object({
  predicted_comments: z.number(),
  comment_rate: z.number().describe("Comments per 1000 views"),
  sentiment_distribution: z.object({
    positive: z.number(),
    neutral: z.number(),
    negative: z.number()
  }),
  comment_themes: z.array(z.object({
    theme: z.string(),
    frequency: z.number(),
    example_comment: z.string()
  })),
  engagement_timeline: z.array(z.object({
    hour: z.number(),
    comment_velocity: z.number(),
    type: z.enum(['initial_spike', 'sustained', 'viral_moment', 'declining'])
  })),
  viral_triggers: z.array(z.string()),
  moderation_alerts: z.array(z.string())
});

const engagement = await generator.generateCustom(
  diverseAudience,
  EngagementSchema,
  `Analyze comment engagement patterns for this content: ${JSON.stringify(contentPiece)}`
);
```
