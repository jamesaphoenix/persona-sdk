title: Persona Class
description: API reference for the Persona class
---

## Constructor

```typescript
new Persona(name: string, attributes: PersonaAttributes)
```

### Parameters

- `name` - The persona's name
- `attributes` - Object containing persona attributes

## Properties

```typescript
readonly id: string        // Unique identifier  
readonly name: string      // Persona name
readonly age: number       // Age
readonly occupation: string // Occupation
readonly sex: Sex          // Gender
```

## Methods

### getAttribute

```typescript
getAttribute(key: string): any
```

Get a custom attribute value.

### setAttribute  

```typescript
setAttribute(key: string, value: any): void
```

Set a custom attribute.

### clone

```typescript
clone(newName?: string): Persona
```

Create a copy of the persona with optional new name.

### toJSON

```typescript
toJSON(): PersonaData
```

Export persona data as JSON.

## Example

```typescript
const persona = new Persona('John Doe', {
  age: 30,
  occupation: 'Engineer', 
  sex: 'male'
});

// Set custom attributes
persona.setAttribute('location', 'New York');
persona.setAttribute('skills', ['JavaScript', 'Python']);

// Clone with new name
const clone = persona.clone('Jane Doe');

// Export as JSON
const data = persona.toJSON();
```
