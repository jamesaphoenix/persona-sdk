## Installation

Install the Persona SDK using your preferred package manager:

```bash
npm install @jamesaphoenix/persona-sdk
```

```bash
yarn add @jamesaphoenix/persona-sdk
```

```bash
pnpm add @jamesaphoenix/persona-sdk
```

## Requirements

- **Node.js**: Version 18 or higher
- **TypeScript**: Version 5.0 or higher (for TypeScript projects)

## Environment Setup

For AI-powered features, you'll need an OpenAI API key:

```bash
# .env.local
OPENAI_API_KEY=your_openai_api_key_here
```

## Import the SDK

### ES Modules (Recommended)

```typescript
import { 
  Persona, 
  PersonaBuilder, 
  PersonaGroup,
  NormalDistribution 
} from '@jamesaphoenix/persona-sdk';
```

### CommonJS

```javascript
const { 
  Persona, 
  PersonaBuilder, 
  PersonaGroup,
  NormalDistribution 
} = require('@jamesaphoenix/persona-sdk');
```

## Verify Installation

Create a simple test to verify everything is working:

```typescript
import { PersonaBuilder } from '@jamesaphoenix/persona-sdk';

const persona = PersonaBuilder.create()
  .setName('Test User')
  .setAge(25)
  .build();

console.log(`Created persona: ${persona.name}, age ${persona.age}`);
// Output: Created persona: Test User, age 25
```
