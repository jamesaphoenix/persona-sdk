# Persona SDK

A lightweight TypeScript SDK for creating personas with AI-powered features:

## Quick Start

Install the SDK:

```bash
npm install @jamesaphoenix/persona-sdk
```

Create your first persona:

```typescript
import { PersonaBuilder } from '@jamesaphoenix/persona-sdk';

// Create a basic persona
const persona = PersonaBuilder.create()
  .setName('Alex')
  .setAge(28)
  .setOccupation('Developer')
  .setSex('other')
  .build();

console.log(persona.name); // "Alex"
console.log(persona.age);  // 28
```
